* Post-install: Secure Boot (lanzaboote)
#+begin_src sh :eval no
sbctl status
sudo sbctl create-keys
sudo sbctl enroll-keys --microsoft
sudo nixos-rebuild switch
sbctl verify
#+end_src

Reboot into BIOS, enable Secure Boot, boot again.
